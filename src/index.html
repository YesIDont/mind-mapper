About

My name is Mateusz Fryc, I'm a programmer based in Poland and huge fan of tools that support problem solving. I've created Free Mind Mapper as I needed good mind mapping tool that wouldn't cost me 100$ month. Originally I've been using it on my desktop but since I've build it using web technologies I thought I could share it.

Have fun! It's for free and there are no strings attached, no data collection, no advetisement. If you would like to support me in making this kind of free apps feel free and hit the big red "Donate" button at the top of the website.

The website is using cokies (your browser memory) to store your settings.

FAQ

Is Free Mind Mapper really for free?
Yup, no fees, no limitations, no data collection, no advertisement, no strings attached whatsoever. If you want to help me to develop more free tools you can always support me by making a dontation.
What if I have an idea on how to improve Free Mind Mapper?
Please, shere it with me by sending email on mateuszfryc@gmail.com. I'm always happy to talk about new functionalities. Couple of those suggestions have already been implemented and my to-do list is always open.
